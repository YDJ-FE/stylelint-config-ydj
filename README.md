# stylelint-config-ydj
the stylelint config for YDJ-FE TEAM


## css config

add this code into your `.stylelintrc.js`

```javascript
module.exports = {
  extends: ['stylelint-config-ydj']
};
```

## scss config

add this code into your `.stylelintrc.js`

```javascript
module.exports = {
  extends: ['stylelint-config-ydj/scss']
};
```
